# Part 1 - Master Roadmap

## Phase 1 (Week 1-2): Foundation and Data Design
1. Setup local environment and coding standards.
2. Design initial database schema:
   - users
   - teachers
   - posts
   - comments
   - votes
   - flags
3. Bootstrap Go Fiber project structure.
4. Connect MySQL and validate migration strategy.

## Phase 2 (Week 3-5): UI Component Baseline
1. Define global CSS tokens for spatial/glass visual system.
2. Integrate GSAP and Lenis.
3. Build Honor page as quality baseline:
   - floating glass cards
   - 3D tilt interactions
   - animated counters

## Phase 3 (Week 6-8): Core API and Frontend Integration
1. Implement auth and JWT flow.
2. Build CRUD APIs for forum and profiles.
3. Connect frontend to API via Axios.
4. Render dynamic counters and card feeds.

## Phase 4 (Week 9-11): Internal Algorithms and AIOps
1. Implement badge score algorithm.
2. Implement weighted vote + vote freeze logic.
3. Build internal moderation service.
4. Integrate claim profile and appeal flows.

## Delivery Strategy
- Work in large parts only.
- Finish each part with docs + code + checklist.
- Wait for "tiep" before next part.

## UX Baseline Details
- Spatial background layers + soft ambient gradients.
- Glass cards with blur, border stroke, and depth shadow.
- Staggered page-load motion powered by GSAP.
- Smooth scrolling initialized by Lenis.
- 3D interaction cards powered by VanillaTilt.
- Odometer counters used in Honor metrics strip.

## Mobile/Desktop Readiness
- Responsive grid behavior at tablet and mobile breakpoints.
- Sticky topbar adapts from pill layout to compact layout.
